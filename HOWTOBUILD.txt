Tools that are used:
    Run and test helpers:
        - air.  repo https://github.com/air-verse/air
        - direnv (.envrc file)
        - Golang migrate. repo https://github.com/golang-migrate/migrate
        - Docker compose
    Libraries:
        - Golang Chi repo https://github.com/go-chi/chi
        - PQ. Postgres lib
        - Request struct validator. repo https://github.com/go-playground/validator
        - swagger. repo https://github.com/swaggo
        - zap. Tool for logging. repo https://pkg.go.dev/go.uber.org/zap
        - uuid. Used for token generation. repo "github.com/google/uuid"
        
        
How to run:
	regular:
		go1.25 run cmd/api/*.go
	with air:
		air
	start Postgres DB in container for app:
		docker-compose up --build
    swagger doc location:
        http://localhost:3000/v1/swagger/index.html
		 
DB operaions(Makefile commands)
	make migration fileName
	migrate-up
	migrate-down
